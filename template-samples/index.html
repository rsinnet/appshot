🚀 Quick Start

1

Install Appshot

Install globally via npm:
npm install -g appshot-cli

2

Quick Start

Interactive setup with templates:
appshot quickstart

3

Add Screenshots

Copy your screenshots to:
screenshots/iphone/

4

Build

Generate final screenshots:
appshot build

🎯 How Templates Adapt to Each Device

📱
iPhone
Fixed 20/80 layout zones with centered device placement. Font size scales from output resolution.
📱
iPad
18/82 layout zones tuned for larger screens with centered placement and larger text scaling.
⌚
Apple Watch
25/75 layout zones with 2-line caption limit and conservative text scaling for readability.
💻
Mac
15/85 layout zones with larger fonts and centered placement optimized for desktop aspect ratios.
